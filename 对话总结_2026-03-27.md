- 更新了 `scripts/track.py`，使其：

### 用户目标

### 新增脚本
- `scripts/gradcam_video.py`
  - 对 JDE 模型做真实反向传播 CAM

  - 默认评分通道：`score-index=4`
  - 叠加输出到视频（`COLORMAP_JET`）
- `scripts/gradcam_image.py`
  - 对单张图片生成 Grad-CAM 叠加图

  - `pred`: `[1, 133, 13566]`
  - 多尺度特征列表（3个尺度）
- 模块末层为 `JDE` 头，网络序列模块索引为 `0..23`。

### 结果

## 5) Git/GitHub 问题与说明

### 远端 push 403 问题
- 现象：`origin` 已存在，且实际仍推送到旧仓库地址，导致权限拒绝（403）。
- 建议处理：
  - `git remote -v`
  - `git remote set-url origin https://github.com/Lansher/YOLO11-JDE.git`
  - `git push -u origin main`

### fetch 与 push 区别
- `fetch`：仅拉取远端更新到本地引用，不改当前工作分支内容。
- `push`：把本地提交上传到远端仓库，需要权限与认证通过。
